# projekt_swa
webovku na autosalon moderních a běžných aut
## Hlavní sekce
- **index.html** - Úvodní stránka webu (Domů)
- **auta.html** - Celková nabídka vozidel (rozcestník)
- **onas.html** - Informace o společnosti a týmu
- **dostupna.html** - Katalog dostupných skladových vozů
- **luxusni.html** - Katalog luxusních a sportovních vozů

## Detaily konkrétních vozidel
- **camaro.html** - Chevrolet Camaro
- **corvette.html** - Chevrolet Corvette
- **cybertruck.html** - Tesla Cybertruck
- **huracan.html** - Lamborghini Huracán
- **passat.html** - Volkswagen Passat
- **q5.html** - Audi Q5
- **escape.html** - Ford Escape

## Funkční soubory
- **process.php** - Zpracování kontaktních formulářů a odesílání poptávek
