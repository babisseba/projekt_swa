# AuraMarine - Dokumentace projektu (SWA)

## Hlavní sekce
- **index.html** - Úvodní stránka webu (Domů)
- **nabidka.html** - Celková nabídka plavidel (rozcestník na čluny a jachty)
- **onas.html** - Informace o společnosti, týmu a naše vize
- **cluny.html** - Katalog dostupných motorových člunů značky Saver
- **jachty.html** - Katalog luxusních a sportovních jachet Sunseeker

## Detaily konkrétních plavidel
- **saver-660-walk-around.html** - Detail člunu Saver 660 Walk Around
- **saver-690-cabin-sport.html** - Detail člunu Saver 690 Cabin Sport
- **saver-870-cabin.html** - Detail člunu Saver 870 Cabin
- **sunseeker-95-quantum.html** - Luxusní jachta Sunseeker 95 Quantum
- **sunseeker-manhattan-66.html** - Luxusní jachta Sunseeker Manhattan 66
- **sunseeker-predator-62.html** - Luxusní jachta Sunseeker Predator 62

## Funkční soubory
- **process.php** - Backend skript pro zpracování kontaktních formulářů a odesílání poptávek přes Brevo API
- **styles/style.css** - Hlavní soubor se styly a design systémem
- **js/main.js** - Skript pro obsluhu navigace a mobilního menu
- **DOKUMENTACE.md** - Kompletní technický popis fungování webu
